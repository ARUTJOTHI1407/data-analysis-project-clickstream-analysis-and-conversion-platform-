##Project Description
This project analyzes website clickstream data to understand user behavior and conversion patterns. Interactive dashboards are built using Power BI to visualize visitor activity, engagement, and revenue insights.

##Objectives
--Analyze user navigation and behavior
--Identify conversion trends across months
--Compare new and returning visitors
--Study engagement using pages visited, clicks, and time spent
--Understand factors influencing purchases

Dataset
**The dataset includes anonymized website interaction data such as:
**Visitor Type (New / Returning)
**Month
**Pages Visited
**Time Spent (seconds)
**Number of Clicks
**Revenue (Purchased / Not Purchased)
**Weekend Indicator

##Tools Used
--Power BI
--Python (Pandas, NumPy)
--Excel / CSV

##Key Insights
--Returning visitors generate most conversions
--Higher engagement leads to higher purchase probability
--Peak conversions occur in specific months
--Weekday traffic is higher than weekend traffic
